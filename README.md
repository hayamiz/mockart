 Mockart
=========

Mockart is a simple frame work for mocking functions in C programming
language.

 Prerequisites
===============

  * GLib 2

 How to build
==============

    $ ./configure
    $ make

 For developers
================

You need autotools and libtool to build. In addition, you need Cutter
testing framework if you want to run tests.

 How to build
--------------

    $ ./autogen.sh
    $ ./configure
    $ make
    $ ./test/run-test.sh
